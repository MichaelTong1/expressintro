After going through the basics of Node.js, 
this will be my introduction to Express.js. 

This web app will ask if you like dogs. 
If yes, you will get to choose from a list of categories!

Created an instance via AWS.

Created on May 17th, 2018.


New website, created on November 9th: 
www.MichaelTong.Me

To start, run the command:
nodemon app.js

If the server is already running on the port, use this command:
fuser -k 3000/tcp

To run indefinitely, run the command:
pm2 start app.js
